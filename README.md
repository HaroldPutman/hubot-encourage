# hubot-encourage

> Robot is very encouraging

## Installation
In a hubot project repository, run:

```
npm install hubot-encourage --save
```
Then add **hubot-encourage** to your `external-scripts.json`:

```json
[
  "hubot-taboo-topics"
]
```
